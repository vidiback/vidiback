1. 개인정보처리방침의 범위

본 개인정보처리방침은 모그레빅스(Mogrevix)가 제공하는 Vidiback 모바일 애플리케이션, 웹사이트, 고객지원 및 관련 서비스에 적용됩니다. Vidiback을 사용하면 본 방침에 따라 정보가 수집, 이용, 저장, 이전, 공개 또는 처리될 수 있습니다. 본 방침에 동의하지 않는 경우 앱 사용을 중단해야 합니다.

본 방침은 Vidiback 이용약관의 일부이며, 특정 기능에서 별도 고지 또는 권한 동의 화면이 제공되는 경우 해당 고지도 함께 적용됩니다.

2. 수집 또는 처리하는 정보

Vidiback은 사용자가 직접 제공하는 정보와 앱 사용 과정에서 자동으로 생성되는 정보를 처리할 수 있습니다.

2.1 사용자가 제공하거나 선택하는 정보

  • 사용자가 가져오거나 편집하는 사진, 영상, 오디오 및 프로젝트 파일
  • 텍스트, 자막, 스티커, 효과, 필터, 배경 제거 결과, 내보내기 결과물 등 사용자가 생성한 콘텐츠
  • 음성 녹음 기능 사용 시 마이크로 입력되는 오디오
  • 고객지원 문의 시 제공하는 이메일 주소, 문의 내용, 첨부 파일, 기기 및 앱 상태 설명
  • 구독 또는 구매 복원 과정에서 처리되는 구매 상태, 상품 ID, entitlement 정보

2.2 앱 사용 중 자동으로 처리되는 정보

  • 앱 실행, 화면 이동, 기능 사용, 내보내기 설정, 광고 시청 여부 등 사용성 분석 정보
  • 크래시, 오류, 성능 문제, 로그, 앱 버전, OS 버전, 기기 모델, 네트워크 상태 등 진단 정보
  • 광고 식별자, 광고 노출/클릭/보상 이벤트, IP 주소, 기기 및 네트워크 정보
  • 프로젝트 썸네일, 캐시, 프록시, 다운스케일 파일, 임시 렌더링 파일
  • 자동 자막, 번역, 배경 제거, 세그멘테이션, 모션 추적 등 ML 기능을 위해 생성되는 분석 결과와 임시 파일

2.3 위치 정보

Vidiback은 현재 위치 기반 기능을 제공하지 않으며, 위치 권한을 요청하지 않고 위치 정보를 의도적으로 수집하지 않습니다. 향후 위치 기반 기능을 제공하는 경우 별도 고지와 권한 동의를 통해 처리합니다.

3. 미디어, 얼굴, 음성 및 ML 처리

Vidiback은 영상 편집 앱이므로 사용자가 선택한 사진, 영상, 오디오를 처리합니다. 배경 제거, 세그멘테이션, 자동 자막, 번역, 모션 추적, 필터 또는 효과 기능은 미디어의 시각/음성 정보를 분석할 수 있습니다.

Vidiback은 이러한 정보를 사용자를 식별하기 위한 목적으로 사용하지 않습니다. 기능에 따라 처리는 기기 내에서 수행되거나, Google ML Kit, Firebase, RevenueCat, AdMob 등 제3자 SDK 또는 서비스와 연동될 수 있습니다. 외부 API 또는 클라우드 처리가 필요한 기능을 출시하는 경우, 해당 제공자와 전송되는 데이터 범위를 본 방침 또는 별도 고지에 반영합니다.

4. 정보 이용 목적

Vidiback은 정보를 다음 목적으로 이용합니다.

  • 미디어 가져오기, 편집, 저장, 복원, 내보내기 기능 제공
  • 자동 자막, 번역, 배경 제거, 모션 추적, 필터, 효과, 전환, 오디오 편집 기능 제공
  • 프로젝트 목록, 썸네일, 캐시, 프록시, 다운스케일 파일 관리
  • 앱 안정성, 크래시 분석, 오류 수정, 성능 개선
  • 사용성 분석, 기능 개선, 품질 관리
  • 광고 표시, 보상형 광고 처리, 광고 성과 측정 및 부정 이용 방지
  • 구독 상태 확인, 프리미엄 기능 제공, 구매 복원
  • 고객지원, 문의 대응, 공지 또는 서비스 관련 안내
  • 보안, 사기 방지, 권리 보호, 분쟁 해결
  • 법령, 스토어 정책, 규제 의무 준수

5. 기기 권한

Vidiback은 다음 권한을 요청할 수 있습니다.

  • 사진 및 동영상 접근 권한: 사용자가 선택한 미디어를 가져오고 편집하기 위해 사용합니다.
  • 오디오 접근 권한: 오디오 파일을 가져오고 편집하기 위해 사용합니다.
  • 마이크 권한: 사용자가 녹음 기능을 사용할 때만 사용합니다.
  • 저장소 권한: 구형 Android 버전에서 미디어 파일을 읽거나 저장하기 위해 사용합니다.
  • 네트워크 권한: 광고, 분석, 크래시 보고, 구독 확인, 번역 또는 외부 서비스 연동에 사용합니다.
  • 포그라운드 서비스 권한: 내보내기 또는 장시간 미디어 처리 작업을 안정적으로 수행하기 위해 사용합니다.
  • 진동 또는 알림 관련 기능: 작업 상태 안내 또는 사용자 피드백에 사용될 수 있습니다.

6. 제3자 서비스

Vidiback은 앱 제공을 위해 다음 제3자 SDK 또는 서비스를 사용할 수 있습니다.

  • Google Firebase Analytics: 앱 사용 분석
  • Google Firebase Crashlytics: 크래시 및 오류 진단
  • Google AdMob: 광고 표시 및 광고 성과 측정
  • RevenueCat: 구독 확인, 구매 처리, entitlement 관리
  • Google ML Kit 및 Google Play Services ML Kit: 번역, 세그멘테이션 또는 기타 ML 기능
  • ONNX Runtime, Sherpa ONNX, LiteRT, OpenCV: 기기 내 ML 또는 미디어 처리 기능
  • Google Play, Google Play Billing, Google Play Services 및 Android 시스템 구성요소

제3자 서비스는 각자의 개인정보처리방침에 따라 정보를 처리할 수 있습니다. 광고, 분석, 크래시, 구독, 번역 또는 ML 기능을 사용할 때 일부 정보가 해당 제공자에게 전송될 수 있습니다.

7. 정보 공유 및 공개

Vidiback은 다음 경우 정보를 공유하거나 공개할 수 있습니다.

  • 앱 기능 제공에 필요한 제3자 SDK 또는 서비스 제공자에게 제공하는 경우
  • 광고 네트워크, 분석 제공자, 결제/구독 관리 제공자와 필요한 정보를 공유하는 경우
  • 법령, 법원 명령, 정부 요청에 따라 필요한 경우
  • 보안, 사기 방지, 기술 문제 해결, 권리 보호를 위해 필요한 경우
  • 회사 구조 변경, 합병, 양도, 사업 이전 등과 관련하여 필요한 경우
  • 사용자가 공유 기능을 명시적으로 사용하는 경우

Vidiback은 사용자의 동의 없이 사용자가 만든 미디어 콘텐츠 자체를 판매하지 않습니다.

8. 사용자 선택권

사용자는 다음 방법으로 정보 처리에 대해 선택할 수 있습니다.

  • 기기 설정에서 사진, 동영상, 오디오, 마이크 등 앱 권한 변경
  • Android 광고 설정에서 개인 맞춤 광고 또는 광고 ID 사용 제한
  • 앱 또는 기기에서 프로젝트, 캐시, 내보내기 파일 삭제
  • 앱 삭제를 통한 기기 내 앱 데이터 삭제
  • 고객지원 이메일을 통한 열람, 정정, 삭제 또는 문의 요청

광고 개인화 제한을 선택해도 일반 광고는 계속 표시될 수 있습니다. 서비스 운영상 필요한 공지, 결제, 보안, 고객지원 관련 메시지는 수신 거부가 제한될 수 있습니다.

9. 보관 기간

프로젝트 파일, 캐시, 썸네일, 프록시, 내보내기 파일은 사용자가 삭제하거나 앱을 제거할 때까지 기기 내에 보관될 수 있습니다. 고객지원 문의 기록은 문의 처리와 분쟁 대응에 필요한 기간 동안 보관될 수 있습니다. 분석, 광고, 크래시, 구독 관련 정보는 각 제3자 서비스의 보관 정책에 따라 보관될 수 있습니다.

법령 준수, 분쟁 해결, 보안 조사, 결제 정산, 권리 보호를 위해 필요한 경우 관련 정보는 해당 목적이 달성될 때까지 보관될 수 있습니다.

10. 아동 개인정보

Vidiback은 만 13세 미만 아동을 대상으로 하지 않습니다. 사용자의 국가 또는 지역에서 더 높은 연령 기준이 적용되는 경우 해당 기준을 따릅니다. 아동의 개인정보가 보호자 동의 없이 수집되었다고 판단되는 경우 vidiback.app@gmail.com으로 연락해 주십시오.

11. 국외 이전

Firebase, AdMob, RevenueCat 등 제3자 서비스는 정보를 사용자의 국가 외 지역에서 저장하거나 처리할 수 있습니다. 사용자는 앱을 사용함으로써 정보가 대한민국, 미국 또는 기타 국가에서 처리될 수 있음을 이해합니다.

12. 보안

Vidiback은 합리적인 기술적, 관리적 보호 조치를 통해 정보를 보호하려고 노력합니다. 다만 인터넷 전송, 모바일 환경, 전자 저장 방식은 완전한 보안을 보장할 수 없습니다. 사용자는 자신의 기기, 계정, 원본 파일 및 내보내기 파일을 안전하게 관리해야 합니다.

13. 외부 링크

앱 또는 웹사이트에는 제3자 웹사이트, 스토어, 광고, 결제 또는 고객지원 링크가 포함될 수 있습니다. 해당 제3자의 개인정보 처리에는 그들의 정책이 적용되며, Vidiback은 제3자 서비스의 처리 방식을 통제하지 않습니다.

14. 개인정보처리방침 변경

본 방침은 변경될 수 있습니다. 중요한 변경이 있는 경우 앱 또는 웹사이트를 통해 고지할 수 있습니다. 변경 후 앱을 계속 사용하는 경우 변경된 방침에 동의한 것으로 볼 수 있습니다.

1. 適用範囲

本プライバシーポリシーは、Mogrevixが提供するVidibackモバイルアプリ、Webサイト、カスタマーサポートおよび関連サービスに適用されます。Vidibackを使用することにより、本ポリシーに記載された方法で情報が収集、利用、保存、移転、開示または処理される場合があります。本ポリシーに同意しない場合は、アプリの使用を中止してください。

本ポリシーはVidiback利用規約の一部です。特定の機能で別途通知または権限許可画面が提供される場合、その通知も適用されます。

2. 収集または処理する情報

Vidibackは、ユーザーが直接提供する情報およびアプリ使用中に自動的に生成される情報を処理する場合があります。

2.1 ユーザーが提供または選択する情報

  • ユーザーが取り込みまたは編集する写真、動画、音声、プロジェクトファイル
  • テキスト、字幕、ステッカー、エフェクト、フィルター、背景削除結果、書き出しファイルなどのユーザー作成コンテンツ
  • 録音機能使用時にマイクから入力される音声
  • カスタマーサポートのために提供されるメールアドレス、問い合わせ内容、添付ファイル、端末およびアプリ状態の説明
  • サブスクリプションまたは購入復元のために処理される購入状態、商品ID、権利情報

2.2 自動的に処理される情報

  • アプリ起動、画面表示、機能利用、書き出し設定、リワード広告イベントなどの利用分析情報
  • クラッシュ、エラー、パフォーマンス情報、ログ、アプリバージョン、OSバージョン、端末モデル、ネットワーク状態
  • 広告ID、広告表示/クリック/報酬イベント、IPアドレス、端末またはネットワーク情報
  • プロジェクトサムネイル、キャッシュ、プロキシ、ダウンスケールファイル、一時レンダリングファイル
  • 字幕、翻訳、背景削除、セグメンテーション、モーショントラッキングなどML機能のために生成される解析結果および一時ファイル

2.3 位置情報

Vidibackは現在、位置情報に基づく機能を提供しておらず、位置情報権限を要求せず、位置情報を意図的に収集しません。今後位置情報機能を導入する場合は、必要に応じて通知し許可を求めます。

3. メディア、顔、音声およびML処理

Vidibackは動画編集アプリであり、ユーザーが選択した写真、動画、音声を処理します。背景削除、セグメンテーション、字幕、翻訳、モーショントラッキング、フィルターまたはエフェクト機能は、メディア内の視覚情報または音声情報を分析する場合があります。

Vidibackは、これらの情報を特定の個人を識別する目的で使用しません。機能によって、処理は端末内で行われる場合、またはGoogle ML Kit、Firebase、RevenueCat、AdMobなどの第三者SDKまたはサービスを通じて行われる場合があります。外部APIまたはクラウド処理が必要な機能を提供する場合、関連する提供者およびデータカテゴリを本ポリシーまたは別途通知に反映します。

4. 利用目的

Vidibackは情報を以下の目的で利用します。

  • メディアの取り込み、編集、保存、復元、書き出し
  • 字幕、翻訳、背景削除、モーショントラッキング、フィルター、エフェクト、トランジション、音声編集機能の提供
  • プロジェクト一覧、サムネイル、キャッシュ、プロキシ、ダウンスケールファイルの管理
  • クラッシュ解析、エラー修正、安定性およびパフォーマンス改善
  • 利用状況の理解、機能改善、品質管理
  • 広告表示、リワード広告処理、広告効果測定、不正利用防止
  • サブスクリプション確認、プレミアム機能提供、購入復元
  • カスタマーサポート、問い合わせ対応、サービス関連通知
  • セキュリティ保護、不正防止、権利保護、紛争解決
  • 法令、ストアポリシー、規制上の義務の遵守

5. 端末権限

Vidibackは以下の権限を要求する場合があります。

  • 写真および動画: ユーザーが選択したメディアの取り込みおよび編集
  • 音声ファイル: 音声の取り込みおよび編集
  • マイク: 録音機能を使用する場合のみ
  • ストレージ: 古いAndroidバージョンでメディアファイルを読み書きするため
  • ネットワーク: 広告、分析、クラッシュ報告、サブスクリプション確認、翻訳、外部サービス連携
  • フォアグラウンドサービス: 書き出しまたは長時間のメディア処理を安定して実行するため
  • バイブレーションまたは通知関連機能: 作業状態案内またはユーザーフィードバック

6. 第三者サービス
